C端用户：关心抽奖             使用比较多

3、M运营模块    C用户抽奖模块

4、需求
以c用户抽奖模块
    1、抽奖资格（抽奖时间、抽奖次数、什么情况下才会有抽奖资格）
    2、奖品（实物、优惠券、购物卡、理财金）
    3、奖项概念（概率、库存、限制次数）
    4、薅羊毛（风控）

5、划分不同子域
核心子域：抽奖子域  draw
支撑子域：抽奖资格 qualification、风控子域  risk、奖品子域  prize
